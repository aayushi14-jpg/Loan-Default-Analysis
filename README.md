# Loan Default & Customer Behavior Analysis

## Objective
To analyze customer loan data and identify default patterns using data analytics techniques.

## Tools Used
- Excel
- SQL (MySQL)
- Python (basic)
- Power BI

## Project Workflow
- Cleaned and prepared loan data using Excel and Python
- Imported data into MySQL and performed SQL analysis
- Built an interactive Power BI dashboard
- Derived insights on customer behavior and loan default risk

## Key Insights
- Customers with lower income levels show higher default rates
- Credit history strongly impacts loan repayment behavior
- Longer loan tenures are associated with increased default risk

## Outcome
This project demonstrates practical data analytics skills and business insight generation.
